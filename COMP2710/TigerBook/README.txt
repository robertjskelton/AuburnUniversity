TO COMPILE:

First, cd into the directory the C++ file is in.
Ex: cd Documents/TigerBook

Next, compile the program using gcc.
Ex: g++ skelton_hw1.cpp User.cpp
You must include both .cpp files to be compiled!

This creates an output file of a.out.

To run a.out, type:
./a.out
